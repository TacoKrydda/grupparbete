## Installation & start

---

### 1. Klona projektet

```bash
git clone [https://github.com/TacoKrydda/grupparbete.git](https://github.com/TacoKrydda/grupparbete.git)
cd grupparbete
```

Eller:
* Download ZIP

---

### 2. Installera beroenden:

i rotmappen
```bash
npm install 
```
i Strapimappen
```bash
npm install 
```

---

### 3. Starta utvecklingsservern:
```bash
npm run start
```

### 4. Öppna i webbläsaren (Ctrl+klick)
När servrarna är igång hittar du applikationen på följande adresser:

* Frontend (React): http://localhost:5173

* Backend (Strapi): http://localhost:1337

---

### Teknikstack
* Backend:Strapi (Headless CMS)
* Frontend:React
* Verktyg:Concurrently
